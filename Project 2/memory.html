Memory

Memory is the storage device of computer that stores data either permanently or temporarily.Memory is the most essential element of a computing system because without it computer can’t perform simple tasks. Computer memory is of two basic type – Primary memory / Volatile memory and Secondary memory / non-volatile memory. Random Access Memory (RAM) is volatile memory and Read Only Memory (ROM) is non-volatile memory.

Differences between Primary memory and Secondary Memory
S.n Primary Memory S.n Secondary Memory
1. It is directly connected to the processor. 1. It is indirectly connected to the processor.
2. It is expensive than secondary memory. 2. It is cheaper than primary memory.
3. It has low storage capacity. 3. It has high storage capacity.
4. It is faster than secondary memory. 4. It is slower than primary memory.
5. It is semiconductor memory. 5. It is optical or secondary memory.
6. Examples: RAM, ROM etc. 6. Examples: hard disk, floopy disk, CD-ROM etc.
